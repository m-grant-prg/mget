NAME
multiget.sh - get Lottery history
SYNOPSIS
multiget.sh [OPTION]
DESCRIPTION
Fetches multiple files whose URL's are specified in a text file.
OPTIONS
-e sourceURL	Use 'sourceURL' as EuroMillions input.
-E targetFile	Save EuroMillions output to 'targetFile'.
-g group	Save target file with 'group' owner. Does not persist between invocations. i.e. Ignored by option -p.
-h	Displays usage information.
-l sourceURL	Use 'sourceURL' as Lotto input.
-L targetFile	Save Lotto output to 'targetFile'.
-o owner	Save target file with 'owner' owner. Does not persist between invocations. i.e. Ignored by option -p.
-p	Saves command line options for future use.
-v	Displays version information.
-w	Use Windows CR/LF instead of Unix LF line endings.
EXIT STATUS
0
if OK,

1
if minor problems,

64
error in command line arguments.
