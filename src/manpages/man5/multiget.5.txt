NAME
File '.multiget' - Retained command line parameter file for multiget (Multiple File Download) script.
DESCRIPTION
The file '.multiget' contains one line for each parameter (Key / Value pair).
FILES
File '.multiget'	The parameters are as follows:-
.br
sourcefile=Name of source file containing the URLs to be downloaded
.br
targetdir=Directory where downloaded files will be stored

The file '.multiget' is stored in the users home directory and is hidden.
